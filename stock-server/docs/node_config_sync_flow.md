# 节点配置自动同步流程设计

## 概述

安卓客户端发现服务器后，自动完成服务器能力探测和节点配置同步的全过程。

## 关键设计决策

### 1. 节点 ID 生成策略

| 策略 | 实现 | 优点 | 缺点 |
|------|------|------|------|
| **UUID（推荐）** | `UUID.randomUUID()` | 零权限、多设备独立、卸载即清理 | 换机丢失配置 |
| 设备指纹 | Android ID 哈希 | 不随应用卸载变化 | 恢复出厂会变、部分设备固定值 |
| 账号绑定 | 用户登录系统 | 换机保留配置 | 需要账号系统、复杂 |

**当前选择：UUID**
- 最简单，不需要任何权限
- 符合"轻量级"设计目标
- 换机场景后续可通过账号系统扩展

### 2. 配置同步时机

| 时机 | 行为 |
|------|------|
| **首次发现服务器** | 立即同步（获取默认配置） |
| **每次启动** | 自动同步（获取最新配置） |
| **用户修改配置** | 实时同步（POST 增量更新） |
| **网络恢复** | 检测"待同步"标记，自动重试 |
| **每 5 分钟** | 后台轮询，检查服务器配置是否有更新 |

### 3. 增量更新策略

```
用户修改：关注列表添加了 "000001"

POST /api/node/config
Body:
{
  "watchlist": {
    "stocks": ["600519", "300750", "000001"]
  }
}

服务器合并后完整配置：
{
  "node_id": "android-a1b2c3d4",
  "watchlist": { "stocks": ["600519", "300750", "000001"] },  ← 更新
  "refresh": { "realtime_interval_sec": 5, "auto_refresh": true },  ← 不变
  "display": { "theme": "dark", "decimal_places": 2 },  ← 不变
  ...
}
```

### 4. 离线模式行为

```
服务器离线
    │
    ▼
┌──────────────────────────────────────┐
│ • 使用本地缓存的节点配置             │
│ • 禁用自动刷新（或延长间隔）         │
│ • 显示"离线"状态，提供重试按钮       │
│ • 用户修改的配置标记为"待同步"       │
│ • 网络恢复后自动重试同步             │
└──────────────────────────────────────┘
```

## 接口调用矩阵

| 场景 | 接口 | 方法 | Headers | 预期响应 |
|------|------|------|---------|---------|
| 服务器发现后探测 | `/api/health` | GET | - | 200 + `{status: "ok"}` |
| 获取服务器能力 | `/api/config` | GET | - | 200 + 全局配置 JSON |
| 首次获取节点配置 | `/api/node/config` | GET | `X-Node-ID: {uuid}` | 200 + 默认配置（服务器自动创建） |
| 后续获取节点配置 | `/api/node/config` | GET | `X-Node-ID: {uuid}` | 200 + 已保存配置 |
| 更新关注列表 | `/api/node/config` | POST | `X-Node-ID: {uuid}` | 200 + 合并后完整配置 |
| 更新主题 | `/api/node/config` | POST | `X-Node-ID: {uuid}` | 200 + 合并后完整配置 |
| 重置配置 | `/api/node/config` | DELETE | `X-Node-ID: {uuid}` | 200 + 默认配置 |

```json
{
  "server": {
    "version": "1.1.0",
    "start_time": "2026-06-25T10:00:00",
    "uptime_seconds": 3600,
    "data_source": "mootdx"
  },
  "sources": {
    "mootdx_tcp": true,
    "tushare": false
  },
  "markets": ["A股", "ETF", "港股"],
  "capabilities": {
    "realtime_single": true,
    "kline_historical": true,
    "intraday_minutes": true,

    "config_query": true
  },
  "limits": {
    "kline_max_days": 500,
    "kline_min_days": 1,
    "intraday_max_minutes": 240
  }
}
```

